# rqsh-web

This is the Next.js landing page and documentation website for the request-shell CLI tool.

## Tech Stack

* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS v4

## Getting Started

### Installation

Make sure you have installed the monorepo root dependencies:

```bash
npm install
```

### Development Server

Run the development server for the website:

```bash
# From the root directory
npm run dev:web

# Or from this folder
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

### Production Build

To build the static production layout bundle:

```bash
# From the root directory
npm run build:web

# Or from this folder
npm run build
```
